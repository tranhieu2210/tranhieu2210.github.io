## Khai báo biến

Sử dụng từ khoá `let` , `const` để khai báo biến


## Sử dụng let

Khai báo biến bình thường 

```javascript
// Khai báo biến
let number;

// khai báo biến và khởi tạo giá trị
let age = 20;

```

## Sử dụng const

Được sử dụng để khai báo hằng số 

```javascript
const number = 10
```

> Không thẻ gán lại giá trị cho 1 biến const
